pygame dependencies on windows

These dependencies are built for Pygame 1.9.2 and Python 2.6 and later. They were built using the msys_build_deps.py script included with the Pygame source files. The dependencies were compiled and linked with MinGW 4.6.1. See the wiki at pygame.org for details. Binaries were stripped, strip --strip-all, to reduce size. They are linked to the Windows C runtime library msvcr90.dll that is installed with Python. The list shows the library versions contained in prebuilt as well as links to their source bundles.
